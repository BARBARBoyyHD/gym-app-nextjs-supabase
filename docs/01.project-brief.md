## 2. Background

Gym saat ini umumnya hanya melakukan pencatatan manual untuk member dan pembelian paket membership. Tidak ada fasilitas digital untuk memberikan nilai tambah seperti materi latihan, edukasi fitness, atau kursus video bagi member.

FitTrack dibuat untuk:

* Memperbaiki proses manajemen member
* Menambah value membership melalui konten pembelajaran fitness
* Menyediakan dashboard admin untuk pengelolaan paket dan kursus dengan mudah

---

## 3. Objective

Membangun aplikasi berbasis web yang memungkinkan admin untuk mengelola paket membership dan kursus video, serta memungkinkan member untuk melihat status membership dan mengakses konten pembelajaran sesuai hak akses.

**Tujuan utama:**

* Sistem demo yang terlihat production-ready untuk dipresentasikan kepada HR, tech recruiter, atau portfolio project.

## 10. Risks & Mitigation

| Risk                             | Mitigation                                       |
| -------------------------------- | ------------------------------------------------ |
| Embed video tidak aman           | Whitelist only: YouTube, Vimeo, Supabase Storage |
| Membership expiry tidak otomatis | Scheduled function / cron on Supabase            |
| Data bocor                       | Implement RLS policies per-table                 |
| Deployment error                 | Environment variables dipisah (dev vs prod)      |
