# Add Membership Plan CRUD API

## Overview
Implement a complete CRUD API for membership plans following the established patterns in the application. This includes creating API endpoints for creating, reading, updating, and deleting membership plan records with proper handler usage.

## Motivation
The application needs a way to manage membership plans for gym members. This functionality is essential for the business logic to support different membership tiers and pricing structures.

## Scope
- Create API endpoints for membership plans following the standardized directory structure
- Implement GET, POST, PUT, and DELETE operations
- Ensure proper handler usage to maintain consistency with other API endpoints
- Define proper TypeScript types for membership plan entities

## Out of Scope
- Frontend interface for managing membership plans
- Business logic for applying membership plans to members
- Payment processing integration
